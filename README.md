## 악보기호

c, d, e, f, g, a, b, : 음계 (도, 레, 미, 파, 솔, 라, 시)

r : 쉼표

\# : 샵

$ : 플랫

% : 내츄럴

v : 음량

< : 1 옥타브 내림

\> : 1 옥타브 올림

t : 템포

. : 반박자 추가

i : 기본 박자

u{} : 기본 악기

s{} : 기본 샵

p{} : 기본 플랫

+ : 동시음표
